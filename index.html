Learning to Necro

~ Completed sometime 2012

Learning to Necro

One of my first projects. Around the release of Guild Wars 2, I built this website as a way to learn more about html. While it may not serve a real purpose for players in the game today, I can look back at this being a first stepping stone into learning more about web development.

Tribute Page

~ Completed May 05, 2017

Tribute Page

The first basic front end development project I created for freeCodeCamp. I made a tribute page to John M. Keynes a 20th century British economist.

View Site

jQuery Playground

~ Completed May 04, 2017

jQuery Playground

A playground used in several freeCodeCamp challenges teaching about how to use various jQuery commands. While this is not very much of a project to show off, I believe it is a good thing to recognize a simple project that allows you to test the many functions of jQuery.

View Site

Cat Photo App

~ Completed May 03, 2017

Cat Photo App

Simple starting project. Teachings of responsive design were made as well as the basic foundations of html, css, and bootstrap were taught in this lesson/project. This was made to fit the size of a fairly small screen-sized phone.
